**Access Point**

Connect to the WiFi access point `Cactus {UNIQUE_MAC_ADDRESS}`.

Go to the web page `http://cactus.local` to fill in the WiFi credentials and [IFTTT webhook key](https://ifttt.com/services/maker_webhooks/settings).
